#  Sales Performance Analytics Dashboard (Power BI) 

<img src="https://raw.githubusercontent.com/gaml8467-source/Sales-Analytics-Dashboard-Power-BI-/refs/heads/main/powerbi_%20sales/image/Sales%20%26%20Product%20Analysis.png" width="900">

##  Project Overview
This project presents an interactive **Sales Performance Analytics Dashboard** built using **Power BI** to monitor and analyze business sales performance across products, countries, and customer segments.

The dashboard transforms raw transactional sales data into actionable insights about **revenue, profit, costs, discounts, and growth trends** using data modeling, DAX calculations, and interactive visualizations.

It demonstrates a complete **Business Intelligence workflow** from data cleaning to KPI tracking and dashboard storytelling.

---

##  Dataset Information
- Source: Kaggle
- Domain: Sales & Business Analytics
- Records: 700 rows
- Columns: 16 features

### Key Columns
- Segment
- Country
- Product
- Discount Band
- Units Sold
- Manufacturing Price
- Sale Price
- Gross Sales
- Discounts
- Sales
- COGS
- Profit
- Date
- Month Number
- Month Name
- Year (added during cleaning)

---

##  Data Cleaning & Preparation

The following preprocessing steps were performed:

- Checked missing values
- Fixed data types
- Added calculated **Year column**
- Prepared structured data model
- Optimized fields for reporting and filtering

---

##  DAX Measures (KPIs)

Created business KPIs including:

- Total Sales
- Total Profit
- Total Units Sold
- Total COGS
- Total Discount
- Average Sale Price
- Profit Margin %
- Sales Year-over-Year (YoY)
- Top Country

These measures allow dynamic and interactive performance tracking.
